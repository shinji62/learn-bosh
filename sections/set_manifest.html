Deployment manifest specifies what services to deploy, their properties and resources configuration. The manifest contains a placeholder director_uuid that needs replacing.

Update the director_uuid value in the templates/bosh-lite-v2.yml with the value from:

$ bosh status --uuid

Set the deployment manifest:

$ bosh deployment templates/bosh-lite-v2.yml

Deployment set to '..learn-bosh-release/templates/bosh-lite-v2.yml'
